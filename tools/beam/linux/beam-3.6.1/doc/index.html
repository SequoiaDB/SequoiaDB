News

January 2, 2011 -- BEAM 3.6.1 released
This is a maintenance release. It includes many bug fixes from our previous release. It also includes some new multi-threaded programming configuration and complaints. Please read the announcement for a detailed list of changes.

July 18, 2010 -- BEAM 3.6.0 released
This is a major release. It includes a new parser for C and C++, which brings better compatibility for many modern compilers. It also features many bug fixes and ehancements. Please read the announcement for a detailed list of changes.

December 16, 2009 -- BEAM 3.5.3 released
This is a maintenance release. New checks for locking are included with this release, as well as a large number of bug fixes and enhancements. Please read the announcement for a detailed list of changes.

July 12, 2009 -- BEAM 3.5.2 released
This is a maintenance release. We have further improved algorithms for pointer analysis that may result in fewer false positives and/or more complaints that are being reported. The release also includes several bug fixes and some miscellaneous enhancements. Please read the announcement for a detailed list of changes.

December 12, 2008 -- BEAM 3.5.1 released
This is a maintenance release. As usual a large number of bug fixes and tweaks are included. Please read the announcement for a detailed list of changes.

June 25, 2008 -- BEAM 3.5.0 released
This is a major release. It includes several usability enhancements, some new checks, and many bug fixes and tweaks. The BEAM tracing tool now also runs on x86_64. Additionally, BEAM can be invoked to run beam_configure under the covers thereby eliminating the need to store (and potentially mix up) configuration files. Please read the announcement for a detailed list of changes.

April 29, 2008 -- 9.17.*.* subnet is blocked
BEAM download attempts (w3.eda.ibm.com/beam/install.html) that originate from the 9.17.*.* subnet are now being blocked. This is a temporary measure to protect IBM assets. If you are an IBM employee who cannot download BEAM because of this please send a note to the BEAM user mailing list and we will arrange to get you a copy of BEAM by other means.

December 14, 2007 -- BEAM 3.4.2 released
This is a maintenance release. It includes an initial release of a build tracing tool that greatly simplifies integration of BEAM into a build process. As usual a large number of bug fixes and tweaks are included. Please read the announcement for a detailed list of changes.

September 30, 2007 -- BEAM 3.4.1 released
This is a maintenance release. It includes new checks for Java, numerous smaller enhancements and bug fixes. Please read the announcement for a detailed list of changes.
