The beam_sort_log Command

The script beam_sort_log allows the post-processing of files containing complaints. It is is especially useful when BEAM is run in parallel and the same complaint file is used for all runs.

Invocation

The script is invoked like so: 

beam_sort_log options input-file output-file

The script's function is controlled via its command line options.

Option Meaning
--by-file sort complaints by file
--by-type sort complaints by type
--stat write frequency count for complaints to STDOUT
--uniq remove duplicate complaints
--version write version number to STDOUT and exit
