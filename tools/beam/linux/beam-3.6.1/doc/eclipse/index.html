BEAM Eclipse Plug-in

The BEAM Eclipse Plug-in lets Eclipse users run BEAM and view the results from within the Eclipse environment (screenshot). It is similar to the the emacs interface - it will match up BEAM complaints to your source code and allow you to automatically move from one complaint to the next. In addition, the Eclipse Plug-in also lets you run BEAM from within Eclipse IDE easily, automatically extracting information from your project and running BEAM over your source files for you.

This plug-in is built on top of the TPTP Analysis Platform. Those who plan to use this plug-in need to be familier with TPTP Analysis Platform first.

Prerequisites

  • Eclipse on Linux x86 or Windows
  • Eclipse TPTP (http://www.eclipse.org/tptp)

The following table describes which Eclipse and TPTP versions you need to install for which BEAM Eclipse Plug-in version.

Installation

The BEAM Eclipse Plug-in is available through the Eclipse software updates mechanism. Follow these steps to install it:

  1. Launch the Eclipse workbench
  2. Select the Help, Software Updates, Find and Install menu.
  3. Select Search for new features to install and press Next.
  4. Press the 'New Remove Site' button and input https://w3.eda.ibm.com/beam/eclipse/site.xml in the URL field. Input any name to the Name field.

  5. Select the site you just created and press Finish.
  6. Select the appropriate BEAM version as described in prerequites and press Next:

  7. Accept the license terms and press Next.
  8. Press Finish. If you see a dialog entitled Verification, press Install.

Usage

For instructions and help on using the BEAM Eclipse Plug-in, select the Help, Help Contents menu from within the Eclipse workbench. For information about TPTP, you can visit the TPTP Web site.
