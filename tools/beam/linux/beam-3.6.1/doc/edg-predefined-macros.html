Predefined Macros of the C/C++ EDG Parser

The EDG parser predefines only those macros that are required to be defined by the various language standards. They are:

C89 C99 C++
Name Value Name Value Name Value
__DATE__ current date __DATE__ current date __DATE__ current date
__FILE__ current file __FILE__ current file __FILE__ current file
__LINE__ current line __LINE__ current line __LINE__ current line
__TIME__ current time __TIME__ current time __TIME__ current time
__STDC__ 1 __STDC__ 1 __STDC__ 1
__STDC_VERSION__ 199409L __STDC_VERSION__ 199901L __cplusplus 199711L
    __STDC_HOSTED__ 1
