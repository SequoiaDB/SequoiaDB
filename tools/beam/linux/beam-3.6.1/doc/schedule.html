Release Schedule

Due to the research nature of BEAM development we do not provide a release schedule. We release whenever new features become available and when they are robust enough for general consumption.

As usual, we'll be making maintenance releases whenever it appears necessary. Those releases are mainly bug fix releases but typically also include some new features such as additional checks.

If you want to be informed whenever a new BEAM version is released, please subscribe to the BEAM announce mailing list.
