BEAM Quick-start Guide

Contents:

Please note that the instructions are meant to get you started. For build integration you should spend a few minutes reading the build integration guide.

  • Getting and Installing BEAM
  • Configuring BEAM for your compiler
  • Testing BEAM
  • Where to go from here

Getting and Installing BEAM

Follow the instructions on the installation page.

Configuring BEAM for your compiler

Look on the compilers page and find out how to invoke BEAM for your compiler.

Testing BEAM

If you want to be sure that BEAM understands your compiler correctly, use the instructions you obtained from the compilers page to invoke BEAM on this file. You should see these complaints.

Where to go from here

Here are some places you may wish to go next:

  • For frequently asked questions and answers, see the BEAM FAQ.
  • To customize BEAM behavior and complaints, see the customization guide and the function attributes.
  • To learn about viewing BEAM complaints with EMACS, which will automatically bring up the source files and color the relevant code, see the emacs guide.
  • For help with integrating BEAM into your builds see the build integration guide.
  • For other information, simply browse the BEAM Homepage.
