BEAM and CTE

BEAM is installed in CTE.

This installation supports the Linux-x86 and AIX-5.1.x platforms for C and C++.

Java is not supported in CTE at this time - if you would like to use BEAM for Java, please install beam locally.

The path to BEAM is $cte/tools/beam, where $cte is the path to your local CTE mirror (/afs/eda/cte, for example).

Inside that directory are the supported versions of BEAM as well as a prod symlinks. The prod symlink points to the newest version of BEAM installed in CTE.

We recommend that users run BEAM via the prod symlink so they are always running the latest and greatest: 

/afs/eda/cte/tools/beam/prod/bin/beam_compile ...

After each BEAM release, the latest version is automatically added to CTE and should be available at your local CTE mirror within a day or two.

Only a maximum of 2 versions will be supported in CTE. Once a new version of BEAM is released, any versions older than the most recent 2 will be deleted.
