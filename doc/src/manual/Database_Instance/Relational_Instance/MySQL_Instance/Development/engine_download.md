##MySQL 版本要求##

MySQL 5.7.24+

##MySQL JDBC 驱动下载##

用户可以从 [maven][maven_jdbc] 仓库或者 [MySQL 官网][official_jdbc] 下载合适版本。

##MySQL ODBC 驱动下载##

用户可以在 [MySQL ODBC][mysql_odbc] 官方下载页面中选择合适的系统和版本进行下载。
