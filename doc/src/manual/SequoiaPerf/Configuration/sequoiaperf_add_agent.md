    目录名：添加Agent服务

1. 进入【Sequoiaperf 配置】页面后选择【SequoiaPerf 配置】选项卡，点击 **添加 - Agent 服务** 按钮并填写表格

 ![添加Agent服务][add_agent_service]


    - 名称：选择要添加的 Agent 服务
    - 主机 IP：Agent 服务所在的主机 IP，不能填写主机名或 127.0.0.1
    - 端口：Agent 服务端口号


2. 点击 **提交** 按钮完成 Agent 服务的添加

![添加成功][add_agent_service_succeed]
