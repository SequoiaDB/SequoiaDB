     目录名称：SequoiaDB慢查询对应的访问计划详情

SequoiaDB 慢查询对应的访问计划详情，可以通过 SequoiaDB 慢查询语句访问计划细节页面查看。该页面包含如下
内容：

- AccessPlanID: 访问计划 ID
- CollectionSpace: 集合空间
- Collection: 集合
- AccessCount: 访问数量
- CacheLevel: 缓存级别
- RefCount: 引用数量
- ScanType: 扫描类型
- Score: 分数
- Sort: 排序
- Hint: 提示
- IndexName: 索引名称
