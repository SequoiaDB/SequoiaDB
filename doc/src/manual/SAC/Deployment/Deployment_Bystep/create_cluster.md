SAC 的集群是主机资源集群，用于安装分布式存储、数据库实例等服务。

1. 点击右下方 **创建集群** 按钮

   ![部署][create_cluster_1]

2. 根据实际需求修改参数后点击 **确定** 按钮

   ![部署][create_cluster_2]

3. 集群创建成功

   ![部署][create_cluster_3]

4. 进行添加主机操作， [点击查看][add_host]
