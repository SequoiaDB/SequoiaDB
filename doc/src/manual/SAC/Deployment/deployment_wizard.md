
1. 点击右下方 **一键部署** 按钮

   ![部署][deployment_wizard_1]

2. 在提示窗口根据实际需求修改参数后点击 **确定** 按钮

   ![部署][deployment_wizard_2]

3. 进入【添加主机】页面

4. 部署主机，[点击查看][add_host]

5. 部署 SequoiaDB 集群，[点击查看][add_adb_module]

> **Note:**
>
> 首次安装不建议修改默认值，如果参数设置不当，可能会影响服务安装。
