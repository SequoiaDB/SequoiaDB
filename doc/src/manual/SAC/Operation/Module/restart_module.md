本文档主要介绍存储集群的重启操作。具体操作步骤如下：

1. 进入【部署】页面，点击【存储集群操作】->【重启】按钮
![打开重启][restart_1]

2. 选择需要重启的服务，点击 **确定** 按钮
![重启弹窗][restart_2]

3. 重启完成
![重启完成][restart_4]
