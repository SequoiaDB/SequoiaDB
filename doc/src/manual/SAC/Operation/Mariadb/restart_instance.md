本文档主要介绍重启 MariaDB 实例操作。具体操作步骤如下：

1. 进入【部署页面】->【数据库实例】页面，点击【实例操作】->【重启实例】按钮

   ![重启弹窗][restart_1]

2. 选择需要重启的数据库实例，点击 **确定** 按钮

   ![重启弹窗][restart_2]

3. 重启完成

   ![重启完成][restart_4]
