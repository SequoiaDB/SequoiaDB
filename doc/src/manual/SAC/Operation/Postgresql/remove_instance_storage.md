用户可以通过移除实例存储操作解除 SQL 实例与 SequoiaDB 数据库的关联。

1. 进入【部署】->【数据库实例】页面，点击【配置实例存储】->【移除实例存储】按钮
![移除实例存储][pg_sdb_5]

2. 选择需要移除的配置，点击 **确定** 按钮
![移除实例存储][pg_sdb_6]

3. 移除成功
![移除实例存储][pg_sdb_7]
