    webSphere Readme

WebSphere 是 IBM 的网络基础架构软件，其包含的 WebSphere Application Server 是面向企业应用的灵活且高度安全的 Java 服务器运行时环境。

本章文档将从如下方面介绍基于 SequoiaDB 巨杉数据库存储引擎的 WebSphere 服务：

- [连接][connection]
- [Web 应用][web_application]
