
##NAME##

sleep - sleep a few milliseconds

##SYNOPSIS##

**sleep(\<ms\>)**

##CATEGORY##

Global

##DESCRIPTION##

Sleep a few milliseconds.

##PARAMETERS##

ms（ *number*， *Required* ）

Time for sleep, Unit：milliseconds.

##RETURN VALUE##

NULL.

##ERRORS##

NULL.

##HISTORY##

v1.0 and above.

##EXAMPLES##

Sleep 1 second.

```lang-javascript
> sleep(1000)
```
