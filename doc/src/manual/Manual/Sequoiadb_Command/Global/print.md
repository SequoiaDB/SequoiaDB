
##名称##

print - 将输入内容输出到标准输出

##语法##

**print(\<contents\>)**

**println(\<contents\>)**

##类别##

Global

##描述##

将输入内容输出到标准输出。print 函数不会换行，println 函数会换行。

##参数##

contents（ *string*， *必填* ）

要输出的内容

##返回值##

无

##错误##

无

##版本##

v1.0 及以上版本

##示例##

格式化输出文件

```lang-javascript
> println("hello world")
hello world
 ```
