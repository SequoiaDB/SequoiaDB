SdbCursor 类主要用于操作游标，包含的函数如下：

| 名称 | 描述 |
|------|------|
| [close()][close] | 关闭当前游标 |
| [current()][current] | 获取当前游标指向的记录 |
| [next()][next] | 获取当前游标指向的下一条记录 |
| [arrayAccess()][arrayAccess] | 将结果集保存到数组中并获取指定下标的记录 |
