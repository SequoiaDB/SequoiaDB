##名称##

stop - 停止当前节点

##语法##

**node.stop()**

##类别##

SdbNode

##描述##

该函数用于停止当前节点。

##参数##

无

##返回值##

函数执行成功时，无返回值。

函数执行失败时，将抛异常并输出错误信息。

##错误##

当异常抛出时，可以通过 [getLastErrMsg()][getLastErrMsg] 获取错误信息或通过 [getLastError()][getLastError] 获取[错误码][error_code]。更多错误处理可以参考[常见错误处理指南][faq]。

##版本##

v2.0 及以上版本

##示例##

停止 node 节点

```lang-javascript
> node.stop()
```
