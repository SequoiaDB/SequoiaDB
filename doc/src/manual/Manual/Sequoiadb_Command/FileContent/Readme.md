FileContent 类主要用于文件内容的操作，包含的函数如下：

| 名称 | 描述 |
|------|------|
| [clear()][clear] | 清除缓冲区内容 |
| [getLength()][getLength] | 获取二进制流的大小 |
| [toBase64Code()][toBase64Code] | 将二进制流转化为 base64 编码格式 |
