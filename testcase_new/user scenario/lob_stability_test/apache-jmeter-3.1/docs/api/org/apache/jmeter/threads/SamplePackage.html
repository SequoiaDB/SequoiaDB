org.apache.jmeter.threads

Class SamplePackage

  • java.lang.Object
    • org.apache.jmeter.threads.SamplePackage

  • public class SamplePackage
extends Object
    Packages methods related to sample handling.
    A SamplePackage contains all elements associated to a Sampler:
    • SampleListener(s)
    • Timer(s)
    • Assertion(s)
    • PreProcessor(s)
    • PostProcessor(s)
    • ConfigTestElement(s)
    • Controller(s)
